### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

Clone the repository:

```bash
git clone https://github.com/marcaosi/conselhos.git
cd conselhos
```

Install the dependencies:

```bash
npm install
# or
yarn
```

### Running the App

Start the development server:

```bash
npm run dev
# or
yarn dev
```

The app will be available at `http://localhost:5173/`.
